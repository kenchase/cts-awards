A comprehensive WordPress plugin for managing awards with advanced search, filtering, and REST API functionality.

== Description ==

CTS Awards provides a complete solution for managing awards and their recipients with advanced search and filtering capabilities.

**Key Features:**
* Custom post type for awards with full WordPress integration
* Award categories taxonomy for organized classification
* Advanced search by recipient name, organization, year, and category
* REST API endpoints for integration
* Responsive, AJAX-powered interface
* ACF integration for flexible data management
* Full internationalization support

**Requirements:**
* Advanced Custom Fields plugin (required dependency)

For detailed documentation, development setup, and API reference, see the [GitHub repository](https://github.com/kenchase/cts-awards).

== Installation ==

1. Install and activate the Advanced Custom Fields plugin (required)
2. Upload the plugin files to `/wp-content/plugins/cts-awards/` or install through WordPress admin
3. Activate the plugin through the 'Plugins' menu in WordPress
4. The Awards post type and category taxonomy will be automatically created
